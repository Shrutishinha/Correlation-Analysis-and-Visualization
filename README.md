## 📌 Overview

This project performs **Correlation Analysis** on the **California Housing Dataset** to discover relationships between numerical features using **Pearson Correlation**, **Heatmaps**, and **Scatter Plots**.

---

## ✨ Features

- 📊 Correlation Matrix
- 🔥 Heatmap Visualization
- 📈 Scatter Plot Analysis
- 🎯 Feature Selection
- 📉 Correlation Interpretation
