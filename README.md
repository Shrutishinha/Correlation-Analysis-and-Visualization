# Correlation Analysis, Visualization, and Interpretation

## Objective
To calculate, visualize, and interpret correlations between variables using the California Housing dataset.

## Techniques Used
- Pearson Correlation Coefficient
- Correlation Matrix
- Heatmap Visualization
- Scatter Plots
- Correlation Interpretation

## Dataset
California Housing Dataset from Scikit-Learn

## Libraries
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Key Findings
- MedInc shows the strongest positive correlation with MedHouseVal.
- Heatmaps provide a visual representation of relationships between features.
- An r-value between 0.50 and 1.00 indicates a strong positive linear relationship.

## Conclusion
Correlation analysis helps identify relationships among variables and supports feature selection for machine learning models.
